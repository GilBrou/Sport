Installation

Back-end

Clone the repository of SportSee back-end:

git clone https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard.git

Inside this repository, install dependencies:

npm install

Launch back-end on port 3000 (default port):

npm run start

Front-end

Clone the repository of SportSee front-end:

git clone https://github.com/GilBrou/gillianBrousse_12_15042022.git

Inside this repository, install dependencies:

npm install

Launch front-end on port 3002:

npm start

Front-end is now rendered at URL http://localhost:3002.

  • There is two existing users :
    • http://localhost:3002/18
    • http://localhost:3002/12
